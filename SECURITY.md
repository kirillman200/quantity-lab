# Security policy

A public security-reporting channel is not configured yet. This policy and the production security page will be updated when a dedicated channel is available. Do not include credentials, private project details, or personal information in unsolicited reports.

Project Quantity Lab is a static, browser-only application. It has no login, server database, or public API. Calculation inputs can be stored in the current browser and encoded into a share link only when the user chooses those actions.

Supported releases are the current production deployment and the latest commit on `main`. Please allow a reasonable remediation window before public disclosure.
